Picetín

Juguete motricidad fina para niños.

Juego didáctico para niños entre 4 y 6 años, con el fin de desarrollar habilidades motoras y motricidad fina mediante el ejercicio de abrochar y desabrochar cordones.

  • Categoría: Diseño textil.

¿Comó cuido mi jardín?

Libro ilustrado para niños.

Libro ilustrado tipo diario de vida basado en el método Montessori, buscando potenciar la independencia y la resolución de problemas en los niños.

  • Categoría: Ilustracion y diseño gráfico.

Komorebi

Pagina web para estudiar y practicar japones.

Servicio web que busca reunir libros para niños en idioma japonés para ayudar en el estudio y practica del idioma. Centrado usuarios adolescentes y adultos, potenciando la participación entre miembros.

  • Categoría: Diseño gráfico y web.

Handy Hobby

set de manualidades para principiantes.

Contextualizado en contexto pandemia, se ideo este proyecto para incentivar a personas mayores de 25 años en iniciar un nuevo hobby en el ámbito de la creatividad, con este kit con manualidades sencillas y fáciles de hacer.

  • Categoría: Packaging.

Origamigo

Pagina web sobre manualidades para niños.

Pagina web pensada para contexto clases infantiles en la que se expliquen diversas manualidades fáciles y entretenidas para niños.

  • Categoría: Diseño Gráfico y web.

Bolso para aletas.

Bolso impermeable doble para nadar, con espacio para aletas.

Bolso pensado para personas que hacen natación con implemento(aletas), separado en dos espacios, uno destinado para los objetos que tienen contacto con el agua, mientras que el otro esta pensado para objetos que deban permanecer secos.

  • Categoría: Diseño textil.
